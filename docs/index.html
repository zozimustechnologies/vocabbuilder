Why spaced repetition works

🧠

Proven by Science

Based on decades of memory research. Progressive intervals optimize how your brain consolidates knowledge from short-term to long-term memory.

📈

3-5x More Effective

Students using spaced repetition retain 3-5 times more information compared to cramming or passive reading.

⏱️

Just 5 Minutes/Day

Short, focused daily sessions are more efficient than marathon study sessions. Consistency beats intensity.

🎯

Targeted Review

Review only the words you struggle with. Once a word is locked in memory, it's done — no wasted effort.

Get started in 30 seconds

1

Install the extension

Get it from the Edge Add-ons store or Chrome Web Store.

2

Open the side panel

Click the VocabBuilder icon in your toolbar. The extension opens in the browser sidebar.

3

Choose your level

Select difficulty (Simple, Medium, Complex, or Competitive) in Settings based on your goals.

4

Start learning

Review your first word. Mark "Got It" when you know it, or "Not Yet" to schedule a review in 1, 3, 7, 14, or 30 days.

For developers

VocabBuilder is built with vanilla JavaScript, pure HTML/CSS, and Manifest V3. No build step, no dependencies, no frameworks — just clean extension code.

🔧 Tech Stack

  • Frontend: Vanilla HTML5, CSS3, ES6+ JavaScript
  • Architecture: Manifest V3 with Service Worker
  • Storage: Chrome Storage API (100% local)
  • Notifications: Chrome Alarms + Notifications APIs
  • No external dependencies

📁 Project Structure

  • background/ — Service worker (spaced repetition, badges, reminders)
  • sidepanel/ — UI (word cards, stats, settings)
  • data/ — Words, badges, dictionary API integration
  • icons/ — Extension icons (16×16 to 128×128)
  • tests/ — Word validation scripts

Quick Start for Developers

  1. Clone: git clone https://github.com/zozimustechnologies/vocabbuilder.git
  2. Open extensions: chrome://extensions (Chrome) or edge://extensions (Edge)
  3. Enable Developer Mode: Toggle in top right (Chrome) or bottom left (Edge)
  4. Load unpacked: Click "Load unpacked" → Select vocabbuilder folder
  5. Start hacking: Edit files, reload extension, see changes instantly
