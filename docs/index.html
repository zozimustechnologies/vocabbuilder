Get started in 30 seconds

Install the extension

Get it from the Edge Add-ons store or Chrome Web Store.

Open the side panel

Click the VocabBuilder icon in your toolbar. The extension opens in the browser sidebar.

Choose your level

Select difficulty (Simple, Medium, Complex, or Competitive) in Settings based on your goals.

Start learning

Review your first word. Mark “Got It” when you know it, or “Not Yet” to review later.

For developers

VocabBuilder is open source. Built with vanilla JavaScript, Manifest V3, and no external dependencies.

🔧

Tech Stack

Frontend: Vanilla HTML5, CSS3, ES6+ JavaScript
Architecture: Manifest V3 with Service Worker
Storage: Chrome Storage API (100% local)

📁

Project Structure

background/ — Service worker
sidepanel/ — UI components
data/ — Words, badges, integrations
icons/ — Extension assets

🚀

Quick Start

Clone the repo, open edge://extensions, enable Developer Mode, click "Load unpacked", and start hacking.
